# Once_Javiera

## Como crear un repositorio
### requisios

- Cuenta en github
- manos (opcional)

### ingresa a tu perfil
- veras una seccion llamad repositorios o repository

- debes darle click izquierdo

### una vez adentro
- deberas darle click izquierdo a el botos NEW o Nuevo

### Ahora crealo
- primero deberas asignarle un nombre, este paso es obligatorio
- luego podras personalisarlo un poco, darle descripcion, Readme y mas, pero eso no lo haremos por ahora
- Lo que si haremos sera poner el repositorio publico.
